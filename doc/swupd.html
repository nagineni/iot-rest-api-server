Software Updater (swupd) API documentation version v1-20161207

/swupd

Software Updater (swupd) interface provides a new way of adding and updating a Linux-based OS.

/swupd get post put delete

get

List all available bundles for the current version of OS.

post

Install new bundle(s). Client can specify the URL of the software update server.

put

Update to latest OS version. Client can specify the URL of the software update server.

delete

Uninstall a bundle.

get /swupd

List all available bundles for the current version of OS.

  • Response

/swupd/search

Interface to search the OS for a binary or library.

/swupd/search get

get

Search OS for a binary or library.

get /swupd/search

Search OS for a binary or library.

  • Request
  • Response

Query Parameters

  • library: (number)

    Pass the value 1 to search paths where libraries are located for a match

    Example:

    1
  • binary: (number)

    Pass the value 1 to search paths where binaries are located for a match

    Example:

    1
  • name: (string)

    A substring of a binary, library or filename

    Example:

    nodejs

HTTP status code 200

Search results.

Body

Type: application/json

Schema:

{
  "id": "intel.iot.rest.api.swupd#",
  "$schema": "http://json-schema.org/draft-04/schema#",
  "title": "Software Update",
  "type": "object",
  "properties": {
    "searchList" : {
      "type": "array",
      "items": {
        "type": "string",
        "description": "Search results"
      }
    }
  }
}

Example:

{
  "searchList": ["iot:/usr/share/clear/bundles/iot", "nodejs-basic:/usr/share/clear/bundles/nodejs-basic"]
}
